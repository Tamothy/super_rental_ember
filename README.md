# Super Rental (One-To-Many Relationships)

#### By Tam Nguyen

## Description

Contains a rental application utilizing Ember framework, implementing one-to-many relationships with reviews to rentals.

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)
* [Ember CLI](http://ember-cli.com/)
* [PhantomJS](http://phantomjs.org/)

## Installation

* `git clone https://github.com/Flores120/super_rental_ember` this repository
* `cd super_rental_ember`
* `npm install`
* `bower install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
